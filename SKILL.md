# ARISE —— 写一句脚本并装入，改写你的第一反应

「第一反应」是你在能思考之前的那**半秒**里做出的事——伸手去拿烟、对伴侣脱口顶回去、恐慌性割肉、心里冒出「你不够好」。意志力够不到那个窗口（它来得太晚）。能改写它的，是**反复念一句你挑好的话**，直到大脑的「熟悉度计数器」把它变成新的默认反应。这一个文件就是完整的方法。

---

## 为什么有效（30 秒）

- **频率计数器**：那半秒靠一个**只数数、不核查**的机制运行。它问的是「这句我听过没有？」，不是「这句对不对？」。计数最高的那句，就成了你的默认反应。
- **路由（routing）是关键**：一句话只有**绕过**意识、抵达计数器，才装得进去。意识会**否决**任何它能评判的东西——感受声明（「我很平静」）、预测（「我会戒掉」）、玄学主张。但它对一个纯粹的**动作没有否决招式**。
- **唯一的自检**：把你写的草稿念出来。**如果念完你更难受 → 路由错了**（那一缩就是否决）。如果只是**觉得有点傻、然后没别的** → 路由对了。

---

## ARISE —— 五个维度

一句脚本有**一个必选项（Action 动作）**和**四个可选维度**。别五个都填——用最少、够用的那几个。

| | 维度 | 状态 | 规则 |
|---|---|---|---|
| **A** | **Action 动作**（主干） | **必选** | 身体在压力下也能做出的行为。小、具体、外人看得见。 |
| **R** | **Reason 理由** | 可选 | 一句简短、诚实的 `因为`，念到第 1000 遍你还认。别用「因为我是 X」（循环）。 |
| **I** | **Identity 身份** | 通常隐含 | 那个「谁」。让动作动词去携带它。若要显式：「我是 X」——**绝不**用「因为我是 X」。 |
| **S** | **Situation 情境** | 可选 | 只有**情境式**脚本才点出线索（「当……时」）。普遍式不用。 |
| **E** | **Emotion 情绪** | 可选（不是槽位） | 把**真实**的感受投进词语、理由、记忆，或说出口的语气里。绝不表演。 |

**每句脚本的三条硬约束**：**现在时 · 一口气说得完 · 你自己的话。**

---

## 怎么写（构造顺序）

1. **先定 Action（动作）。** 那半秒该**做**什么？必须是行为，不是感受。（「我做三次呼吸，问一个问题」，不是「我保持冷静」。）如果一个旁观者看不出它有没有发生，那它还是心情/结果——继续打磨。
2. **决定 Scope（范围）。** 普遍式还是情境式？
   - **普遍式**（「我从不吸烟」「我跑我自己的赛道」）——在**所有**相关时刻都成立。装得更快，还能把上百个触发情境收拢成一个默认反应。**只要诚实，就默认用它。**
   - **情境式**（「账单一到，我 60 秒内打开」）——动作只在某个特定线索下才成立时用。关系类、内在批评类脚本通常用它（触发点不是到处都犯）。
   - ⚠ 别不诚实地宣称普遍式（你明明会发火却说「我从不发火」）——意识会察觉并否决它。
3. **检查 Identity（身份）。** 动作动词是不是已经说清了你是谁？是，就让身份隐含。不是，就先打磨动作，再考虑加显式的「我是……」。
4. **需要时才加 Reason（理由）**——当意识想抬杠，或脚本要扛长线时。简短 + 诚实 + 贴着这个动作。
5. **诚实地投入 Emotion（情绪）**——投在它真实存在的地方（词语 / 理由 / 记忆 / 说出口的语气）。如果没有感受，就中性地念；次数照样累积。

**渐进示例（同一个动作，逐步加维度）：**
- `我从不吸烟。` —— 只有动作，普遍式，身份隐含
- `我从不吸烟，因为我想要干净的肺。` —— + 诚实的理由
- `账单一到，我 60 秒内打开。` —— 情境式
- `她说话时，我先看着她的眼睛再回答，因为不这么做一直在让我失去我的女儿。` —— 情境式 + 带真实感受的理由

## 怎么装入

- **随处重复。** 计数器按**频率算账，不按地点**——走路、洗澡、开车、做事的间隙、睡前都行。别等触发那一刻；触发点只当个早期提醒用。
- **预期「抗议」。** 头 ~100 遍会觉得很蠢——**那份抗议是方法在生效，不是失败**（意识在对一句它拦不住的话喊叫）。继续念。
- **按顺序留意这几个路标**：觉得蠢 → 觉得没感觉 → **烦**（「唉又是这句」= 越过熟悉度门槛，再坚持几周）→ 它开始不请自来 → 新反应抢在旧的前面到。
- **每个脚本按 3–12 个月规划**（中位数约 66 天，最长可到 254 天）。两周什么都证明不了。
- **一次只跑 1–2 个脚本。** 计数器带宽有限。
- **维护。** 装入不是一劳永逸——环境一直在重写你。一旦稳定，一个脚本只需偶尔念几遍（约每年 20 分钟）加上**环境摩擦**（烟从抽屉里拿走、手机放出卧室）。旧反应若冒头，就把频率重新调高。

## 调音杠杆（脚本卡住时）

- **人称**：默认第一人称；高压下用**第二人称**（「你现在不吸烟」）；用**第三人称 / 自己的名字**制造距离（「笑来，先吸一口气，问一个问题」）。
- **语言**：用母语装入的声音（比如童年的内在批评），往往用你的**第二语言**来对抗最有效——它绕开了对方的主场。
- **出声**念（一个人时，分量更重）；不方便时**默念**。

## 两类场景要格外当心

上面的机制到哪里都一样，但在这两种情况下，一句「技术上正确」的脚本仍可能错过对方真正需要的——所以要把那多出来的一句话说出来。

**内在批评 / 自我攻击**（「你本可以做更多」「你不够好」）：
- **承诺是重新排序，不是让它闭嘴。** 旧声音不会消失——几个月后，它只是不再**第一个**到。这点很要紧：期待它彻底安静的人，等它哪天再冒出来，就会以为自己失败、然后放弃。要告诉对方成功真正的感觉：新句子先到，旧的变成背景音。
- **把「标准」和「批评者」分开。** 有用的标准是具体的、指向一个改法的（「这一段还要再过一遍」）；批评者是全局的、羞辱性的（「你什么都做不成」）。只对批评者写脚本——标准留着，它带着真实的信息。
- 问一句「这原本是谁的声音？」是信息，不是追责——它让那个声音更容易被当成一次「被装入的东西」，而不是事实本身。

**愤怒 / 冲突 / 关系：**
- 高唤醒时，第一人称会被反应本身劫持（「我没错啊，我话都出口了」），脚本载不进去。**换成第二人称或你自己的名字**——「〔名字〕，先吸一口气，问一句。」那一点别扭，正是让它通过的距离感。
- 只对**不成比例的**那种呛声写脚本——不要碰那种对真实冒犯的正当警报。压住正当的愤怒是另一种问题；如果有人像是在把每一种愤怒都脚本掉，要点破它。
- 这些脚本的前提，是双方在各自状态好的时候都还想好好过。如果存在虐待或控制，一句「更冷静的反应」脚本只会训练人更顺地吸纳伤害——那是下面的安全边界，不是该写的脚本。

## 脚本卡住了？（5 种失败模式）

1. 它针对的是**感受**，不是动作 → 改写成一个行为。
2. 那个`因为`**不是你真心认得的** → 换一个你确实相信的、更朴素的理由。
3. **底层价值观错了**（脚本压过了一个真实的偏好，让你觉得不像自己）→ 换身份，或干脆放弃这个脚本。
4. **环境没一起改造** → 加摩擦；你正在把次数输给触发点。
5. 目标**超出适用范围**（见下）→ 去找对的帮助。

---

## 什么时候**不要**用这个方法

严重 / 活动期创伤或解离 · 有躯体依赖的成瘾（酒精/阿片类/苯二氮䓬类）· 活动期躁狂、精神病性症状或重度抑郁 · 运动技能的精熟（乐器、竞技）· 任何带虐待或控制的关系（那里的目标是安全，不是更冷静的反应）。这套方法可以**配合**专业帮助——但不替代它。
